# PeggleAtHome---CMPM120
itch.io link: https://maycuneta.itch.io/peggle-at-home

external visual assets were made using a mix of Krita and Photoshop
    ball was drawn in Krita
    pegs were made by taking ball image and shifting the color in Photoshop

experience requirements
    -continuous input: ball bouncing after being launched in player inputted direction
    -discrete input: ball is launched by player using mouse click
    -goal: pegs can only be removed by the ball bouncing around and touching them
    -3+ physics based scenes: 3 levels with different peg layouts
    -other scenes provide context to physics scenes: summary screen at end of each level provides info in form of how many balls players used (how many shots they made)
